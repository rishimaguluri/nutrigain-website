About Sujay

Sujay Nalla is a student at The Ohio State University specializing in UX design and user research. As the UX Design & Research Lead for Nutrigain, Sujay was responsible for translating user needs into elegant, intuitive interfaces that addressed the real pain points identified in user interviews.

Sujay's work ensured that every screen in Nutrigain was designed with the student's journey in mind — reducing cognitive load, making critical information accessible in under 3 taps, and creating a visual language that students would trust and adopt quickly.

Key Contributions to Nutrigain
Conducted and synthesized user interviews with OSU students — identifying the 7 ranked user needs through pairwise comparison
Designed the complete UX flow diagram mapping all user paths through the Nutrigain app
Created hi-fidelity wireframes and concept mockups for all 6 explored concepts
Designed the User Experience Journey map tracking student satisfaction from "Wake Up" to "Eat & Rush to Class"
Led the user persona development for "MAX" — the primary user archetype
